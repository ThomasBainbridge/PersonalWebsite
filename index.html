03 — Projects

Personal projects

001

Bearing Condition Monitor

A vibration-based machine learning pipeline for bearing fault diagnosis and severity assessment using the CWRU dataset. Raw .mat vibration signals are ingested, segmented into fixed windows, and passed through an engineered feature extraction stage covering time-domain statistics and frequency-domain descriptors. A hierarchical classifier then outputs both fault type (normal, inner race, ball, outer race) and fault severity, combining them into a final condition label. Models are validated across unseen operating loads using leave-one-load-out cross-validation, with a command-line interface for assessing individual vibration files.

Python Machine Learning Signal Processing Random Forest XGBoost CWRU Dataset Condition Monitoring

002

UAV Performance & Mission Analysis Tool

A Python-based engineering tool for estimating fixed-wing UAV performance and mission-profile behaviour from preliminary design inputs. Aircraft parameters are defined through YAML config files and fed into a performance engine covering stall speed, drag, power, endurance, and range. The tool models segmented mission profiles — including climb, cruise, loiter, descent, and return phases — with per-segment energy accounting across propulsion, hotel, and payload electrical loads. A Streamlit front end enables interactive trade studies, sensitivity sweeps, and scenario comparisons, with CSV and YAML export throughout.

Python Streamlit UAV Aerodynamics Mission Analysis YAML Trade Studies
2025 ↗
003

Motorsport Lap Performance Analyser

A Python tool for analysing Formula 1 lap telemetry that combines data-driven comparison with a first-principles vehicle dynamics model. Laps are aligned over a common distance basis and automatically segmented into braking zones and corners, with time-loss attribution broken down to phase level and a machine learning pipeline extracting features, clustering corner archetypes, and running cross-validated regression. A separate physics stage reconstructs the track from position data, solves a quasi-steady-state lap-time envelope with friction ellipse and aerodynamic downforce, and quantifies how close each driver got to the car's theoretical limit. A single command produces aligned traces, segment rankings, cluster maps, envelope overlays, g-g diagrams, and a summary report — all driven by YAML config files.

Python FastF1 Telemetry Analysis Machine Learning Clustering Motorsport YAML

04 — University Projects

Academic work

A selection of projects and assingments completed during my Undergraduate and postgraduate study.

Laminar Channel Flow — CFD vs Analytical Study

Comparative study of analytical and numerical solutions for fully developed laminar flow in a 2D channel. An analytical velocity profile was derived from the Navier–Stokes equations and implemented in MATLAB as a benchmark. ANSYS Fluent simulations were then run across three structured mesh densities, two outlet boundary conditions, and first vs second order discretisation schemes. A non-isothermal case was also conducted to examine thermal behaviour within the flow field.

ANSYS Fluent MATLAB CFD Mesh Refinement Heat Transfer Cranfield MSc 80%

Numerical Methods for PDEs & High Performance Computing

Two-part assignment covering finite difference schemes and HPC performance analysis. A Fortran solver was developed to apply four explicit schemes (FTCS, Upwind, Lax-Friedrichs, Lax-Wendroff) to a 1D linear advection equation across two initial value problems, with error norm analysis and CFL number investigation. The second part analysed the UCNS3D CFD solver on the Taylor–Green vortex problem using 2–64 MPI processes, assessing parallel speed-up, efficiency, task breakdowns, and CO₂ footprint for both second and third order schemes.

Fortran Numerical Methods MPI HPC PDEs CFD Cranfield MSc 84%

Hybrid Grid Generation — NASA HiLiftPW-1 Trap Wing

Generation and assessment of a coarse RANS-suitable hybrid unstructured grid for the NASA HiLiftPW-1 trapezoidal wing configuration at Re = 8.5 million, following official workshop gridding guidelines. The surface mesh was built in Pointwise using an advancing front algorithm with targeted edge length clustering around high-curvature regions. Near-wall resolution was achieved using T-Rex prism layer meshing with a y⁺ ≈ 1 target, producing 38 prism layers at a growth rate of 1.2. The final grid totalled ~15.9 million cells and was assessed against skewness, aspect ratio, and orthogonal quality metrics before import into ANSYS Fluent.

Pointwise ANSYS Fluent Grid Generation RANS High-Lift Aerodynamics Boundary Layer Meshing Cranfield MSc 82%

Data Analysis & Uncertainty — Turbulent Channel Flow

Processed DNS time-series data from a turbulent channel flow to compute mean and RMS velocity profiles, benchmarked against the Lee & Moser reference dataset. Spectral analysis was performed using FFT with rectangular, Hann, and Blackman windowing strategies, alongside a moving average filter, to investigate spectral leakage and frequency content. Statistical characterisation of velocity fluctuations was carried out through skewness, kurtosis, PDF estimation, and Pearson correlation analysis, with flow visualisation completed in ParaView using Q-criterion iso-surfaces.

MATLAB ParaView DNS Signal Processing Cranfield MSc 73%

Turbulence Modelling — Virginia Tech Stability Wind Tunnel

A group RANS study simulating turbulent airflow through the high-speed leg of the Virginia Tech Stability Wind Tunnel using ANSYS Fluent, with reference conditions derived from experimental operating points across three Mach numbers. Multiple turbulence closures were compared — including k-ε RNG, k-ω BSL, k-kL-ω, and k-ω SST — alongside curvature correction, corner correction, and energy equation effects to assess their influence on secondary flow features. Grid convergence was quantified using Roache's GCI framework across four mesh refinement levels, and turbulence intensity and Reynolds shear stress profiles were benchmarked against DNS reference data.

ANSYS Fluent RANS k-ω SST GCI / Mesh Convergence Cranfield MSc 71%

Numerical Modelling for Compressible Flows

A Godunov-type finite-volume solver for the compressible Euler equations was extended in Fortran to implement HLL and HLLC approximate Riemann solvers alongside second-order piecewise-linear reconstruction with a minmod limiter, verified against a reference solution across three grid resolutions. Multidimensional benchmark problems — the 2D Riemann problem, Mach 3 forward-facing step, and double Mach reflection — were analysed using UCNS3D, with density, Mach number, and pressure contours used to identify wave structures and limiter activation contours used to compare CWENOZ scheme orders and reconstruction strategies.

Fortran UCNS3D Finite Volume Methods HLLC / HLL / WENO Cranfield MSc 85%
